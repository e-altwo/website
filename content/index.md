## Technologies

- [Quartz](https://quartz.jzhao.xyz/):  a fast static-site generator that transforms Markdown content into fully functional websites;
- [Obsidian](https://obsidian.md/): a Markdown-based note-taking application;
	- Quartz  [natively supports](https://quartz.jzhao.xyz/features/Obsidian-compatibility) Obsidian-flavored Markdown.
- [GitHub Pages](https://pages.github.com/): site hosting from GitHub repos;
	- Quartz [instructions](https://quartz.jzhao.xyz/hosting#github-pages) for hosting on GitHub Pages.
